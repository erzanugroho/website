## 6. Responsive Design

### Mobile (< 768px)
- Stack 2 klasemen table vertically
- Match cards full width
- Bracket simplified (stack vertically)
- Hamburger menu for admin

### Tablet (768px - 1024px)
- Klasemen side by side
- Match cards 2 per row
- Bracket compressed horizontally

### Desktop (> 1024px)
- Full layout as designed
- Match cards 3 per row
- Maximum width 1200px centered
